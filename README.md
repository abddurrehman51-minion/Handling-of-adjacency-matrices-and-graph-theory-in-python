# Graph Theory using Adjacency Matrices
## Overview:

This project implements fundamental concepts of **Graph Theory** using **adjacency matrices in Python**. It demonstrates how graphs can be represented mathematically using matrix structures and analyzed using computational tools.
The project is developed in a **Jupyter Notebook (Google Colab)** and also exported as a Python script for modular usage.

## Concepts:

* Graph representation using adjacency matrices
* Directed and undirected graphs
* Weighted vs unweighted graphs
* Matrix-based graph operations
* Visualization of graph structures using heatmaps

The following Python libraries were used:

* `pandas` → Data handling and structured representation
* `numpy` → Numerical computations and matrix operations
* `matplotlib.pyplot` → Data visualization
* `matplotlib.ticker` → Axis formatting and plot customization
* `IPython.display` → Displaying outputs inside Jupyter Notebook
  
## Visualization

A **heatmap representation of adjacency matrices** was used to visually analyze graph structures.

This helps in:

* Understanding connectivity patterns

## Features:

* Construction of adjacency matrices from graph data
* Support for directed/undirected graph modeling
* Matrix-based computation of graph properties
* Heatmap visualization for better interpretation
* Clean and modular Python implementation
